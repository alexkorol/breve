# Jimothy Privacy Policy

Last updated: July 21, 2026

Jimothy is a local-first study app. Your data stays on your device.

## What Jimothy stores

- Study progress (card schedules, streaks, review counts) is stored only on your device.
- Custom decks you import or create are stored only on your device.
- If you add an AI provider API key in Settings, the key is stored only on your device.

Jimothy has no accounts, no analytics, no ads, and no tracking. The developer operates no server and receives none of your data.

## AI features

AI features (deck generation, answer feedback) are optional and only work if you supply your own API key for a provider (OpenRouter or Anthropic). When you use an AI feature, the relevant card content and your prompt are sent directly from your device to that provider. That data is handled under the provider's own privacy policy:

- OpenRouter: https://openrouter.ai/privacy
- Anthropic: https://www.anthropic.com/legal/privacy

If you never add an API key, nothing ever leaves your device.

## Backups

Progress export produces a JSON file that you save where you choose. Sharing a deck creates a link containing the deck data; it is only shared if you send it to someone.

## Data deletion

Deleting the app deletes all app data. You can also clear progress from Settings.
